# Proyecto de Asistencia Automática con Lector de Barras

Este proyecto utiliza PyQt6 para crear una interfaz gráfica de usuario (GUI) que permite la lectura automática de códigos de barras para la gestión de asistencia en la universidad Continental.

## Características

- **Interfaz Gráfica de Usuario (GUI)**: Utiliza PyQt6 para una experiencia de usuario intuitiva y fácil de usar.
- **Lectura de Códigos de Barras**: Integración con un lector de códigos de barras para capturar y registrar asistencias automáticamente.
- **Gestión de Asistencia**: Almacena y gestiona los registros de asistencia en una base de datos.

## Requisitos

- Python 3.8 o superior
- PyQt6
